# Grecchi Odontologia

Site institucional estático da Grecchi Odontologia, preparado para hospedagem no GitHub Pages.

## Visão geral

Este projeto reúne a versão publicada do site da clínica, com foco em apresentar:

- a estrutura da clínica;
- a equipe;
- os tratamentos oferecidos;
- os canais de contato.

O site é composto por HTML, CSS, JavaScript e ativos estáticos, sem dependência de etapa de build.

## Base original e trabalho realizado

Este site foi adaptado a partir de um trabalho original da Tirabassi em PHP. Essa referência aparece no rodapé das páginas e em ativos como o SVG de crédito institucional.

Sobre essa base original, o trabalho desta versão concentrou-se em:

- adaptação de identidade, conteúdo e contatos para a Grecchi Odontologia;
- manutenção da estrutura institucional de navegação entre páginas estáticas;
- atualização do comportamento responsivo para melhorar a navegação em telas menores;
- implementação de menu mobile em JavaScript, com controle de abertura e fechamento;
- substituição/adequação do slider da home para um funcionamento leve em JavaScript puro;
- organização do projeto para versionamento em Git e publicação automática com GitHub Pages.
